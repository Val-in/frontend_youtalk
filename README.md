# YouTalk Blog Project

A clean, responsive blog layout built with semantic HTML5 and modern CSS3 for the YouTalk platform.

## Project Structure

- `README.md` - Project documentation and overview.
- `blog.html` - The main blog page featuring a grid of article cards, categorization, and a featured post banner.
- `article.html` - A detailed single article view with a table of contents, text layout, embedded quotes, and an adaptive sidebar for booking psychological services.

## Features

- **Semantic HTML5:** Built using clean, modern tags (`<main>`, `<article>`, `<section>`, `<aside>`, `<blockquote>`) to ensure excellent SEO and accessibility.
- **Adaptive Layouts:** Flexible CSS Grid and Flexbox structures that gracefully shift content from desktop multi-column views to single-column stacked layouts on mobile devices.
- **Reusable Components:** Seamless integration and visual consistency with the header and footer structure across all pages.
- **Pure CSS:** No heavy frameworks or JavaScript dependencies—optimized for fast loading and smooth rendering.
